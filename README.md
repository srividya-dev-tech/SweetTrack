# SweetTrack 🍫

SweetTrack is a Java-based chocolate order and inventory management system developed as a mini project.

## Features
- Add chocolates
- Manage inventory
- Store customer orders
- File handling using text files
- Simple menu-driven interface

## Technologies Used
- Java
- OOP Concepts
- File Handling
- VS Code

## Project Structure
- `src/` → Java source files
- `data/` → Stored order and inventory data

## Future Improvements
- Database integration
- GUI/Web interface
- Admin login system
- Sales reports
